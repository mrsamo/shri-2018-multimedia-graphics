# Мультимедиа. Графические эффекты.

Современный HTML предлагает ряд возможностей для работы с мультимедиа: встраивание растровой и векторной графики, создание анимаций, проигрывание звуков и видеороликов. Хорошему разработчику необходимо ориентироваться в этих возможностях.

Для оформления сайта, привлечения внимания или просто создания wow эффектов дизайнеры использует разнообразные графические приемы и анимации. Разработчику необходимо понимать, какие техники есть в его арсенале и что следует выбрать для реализации в каждом конкретном случае. На лекции мы осветили все современные техники, но для истинного просвещения рекомендую ознакомиться с другими материалами по теме.

Для понимания возможностей CSS советую время от времени заглядывать на сайт CSS-Tricks (https://css-tricks.com), полистать книгу CSS Secrets и следить за сайтом (и всей деятельностью) ее автора - прекрасной Lea Verou.

Узнать больше о производительности ваших сайтов (анимаций в частности), новых возможностях браузеров помогут видео и статьи Paul Irish (https://www.paulirish.com) и Paul Lewis (https://aerotwist.com). Эти специалисты из Google специализируются на инструментах разработчика (devtools) и оптимизациях web приложений. Советую следить за видео с конференций Google I/O (https://www.youtube.com/playlist?list=PLNYkxOF6rcICniLJ2rfj0FexlA-9zmJJE), которые касаются темы производительности рендера и отрисовки.

Возможности SVG прекрасно освещают Юля Бухвалова (http://css.yoksel.ru) и Sara Soueidan (http://sarasoueidan.com). Вы легко найдете их красочные видео презентации на YouTube. Как на практике использовать WebGL и создавать 3D графику в web доступно рассказывает Василика Климова (https://vasilika.ru). Хорошие статьи по WebGL доступны также на сайте https://webglfundamentals.org/.

Руководство по работе с HTML5 Canvas можно найти на MDN Web Docs (https://developer.mozilla.org/ru/docs/Web/API/Canvas_API/Tutorial) или учиться на живых примерах с HTML5 Canvas Tutorials (https://www.html5canvastutorials.com). Узнать больше о техниках CSS и JS анимаций, об отличиях в браузерной реализации этих технологий поможет мой доклад, который так и называется: Веб-анимация на JS и CSS (https://www.youtube.com/watch?v=zJxPuXNHx_M).
